# Kontragent – Web

Mobil ilova (React Native / Expo) bilan bir xil dizayn va API ga ega veb-ilova. Vite + React + TypeScript + React Router.

## Tuzilma

- `src/app/` – sahifalar (Login, Dashboard, Orders, Profile, Notifications va boshqalar)
- `src/components/` – Layout va boshqa komponentlar
- `src/contexts/` – Auth context
- `src/services/` – API xizmati

## Ishga tushirish

```bash
cd web
npm install
npm run dev
```

Brauzerda: http://localhost:3001

## Build

```bash
npm run build
```
