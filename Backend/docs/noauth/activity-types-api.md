# NoAuth Faoliyat Turlari API

Auth talab qilinmaydi.

Base URL: `http://localhost:8081/api/v1`

## Endpoint

- `GET /noauth/activity-types`

## Javob

`data`:

```json
[
  {
    "id": 1,
    "name": "Oziq-ovqat",
    "status": "active"
  }
]
```

## Izoh

- Faqat `active` holatdagi faoliyat turlari qaytariladi.
