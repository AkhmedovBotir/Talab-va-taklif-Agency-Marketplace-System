# NoAuth Regions API

Base URL: `http://localhost:8081/api/v1`

Auth talab qilinmaydi.

## Endpointlar

- `GET /noauth/regions`
- `GET /noauth/districts?region_id=`
- `GET /noauth/mfys?district_id=`

## Qisqa tavsif

- Regionlar: hammasi.
- Districtlar: ixtiyoriy `region_id` filtri.
- MFYlar: ixtiyoriy `district_id` filtri.
